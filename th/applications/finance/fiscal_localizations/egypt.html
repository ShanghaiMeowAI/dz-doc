ลงทะเบียนกับ Odoo บนพอร์ทัล ETA ของคุณ¶

  • ความลับของลูกค้า 2

Note

  • ETA ควรให้ชื่อผู้ใช้และรหัสผ่านในการเข้าถึงพอร์ทัลออนไลน์ของพวกเขา

  • รหัสเหล่านี้เป็นความลับและควรเก็บไว้อย่างปลอดภัย

การกำหนดค่าบน Odoo¶

หากต้องการเชื่อมต่อฐานข้อมูล Odoo กับบัญชี ETA Portal โปรดไปที่: menusesection:` การบัญชี -> การกำหนดค่า -> การตั้งค่า -> ETAการตั้งค่าใบแจ้งหนี้อิเล็กทรอนิกส์และการตั้งค่า: guilabel: "ETA Client ID" และ:guilabel:ETA Secret, คุณเรียกคืนเมื่อ: อ้าง:` ลงทะเบียน Odooพอร์ทัล ETA ของคุณ <Egypt/e-inquiring-eta-portal> ตั้งค่าเกณฑ์การออกใบแจ้งหนี้ถ้าจำเป็น

Important

  • ** ทดสอบบนพอร์ทัลก่อนการผลิต ** ก่อนที่จะเริ่มออกใบแจ้งหนี้จริงบนพอร์ทัลการผลิต ETA

  • ** ใบรับรองสำหรับสภาพแวดล้อมก่อนการผลิตและการผลิต ** แตกต่างกันเมื่อคุณย้ายจากสภาพแวดล้อมหนึ่งไปยังอีกสภาพแวดล้อมหนึ่ง โปรดตรวจสอบให้แน่ใจว่าได้อัปเดตสิ่งเหล่านี้บน Odooอีกอัน

  • หากยังไม่แล้วเสร็จ กรุณากรอกข้อมูลบริษัทของท่าน และแนบข้อมูลบริษัทของท่านให้ครบถ้วนที่อยู่ ประเทศ และหมายเลขภาษี

  • ข้อมูล บริษัท:

    • เลขประจำตัวผู้เสียภาษีนิติบุคคล

    • หมายเลขสาขา

สาขา¶

สร้างรายชื่อผู้ติดต่อและบันทึกประจำวันสำหรับแต่ละสาขาของ บริษัท และกำหนดค่าการตั้งค่า ETA

เมื่อต้องการทำเช่นนี้ไปที่: ส่วนเมนู: "การบัญชี -> การกำหนดค่า -> บันทึกประจำวัน",แล้วคลิกที่: guilabel:`สร้าง'

ตั้งชื่อบันทึกประจำวันตามสาขาของ บริษัท ของคุณและตั้งค่า: guilabel: "ประเภท" คือ: guilabel: "ขาย" ถัดไป เปิด: menuselection: แท็บ "การตั้งค่าขั้นสูง" แล้วกรอก: guilabel: "อียิปต์"ส่วนการตั้งค่า ETA:

  • ในช่อง: guilabel: "สาขา" ให้เลือกผู้ติดต่อของสาขาหรือสร้าง

  • การตั้งค่า: guilabel: รหัสกิจกรรม ETA

  • การตั้งค่า: guilabel: "รหัสสาขา ETA" (หากมีเพียงสาขาเดียวให้ใช้ "0")

Important

ผู้ติดต่อที่เลือกในช่อง: guilabel: "สาขา" จะต้องตั้งค่าเป็น: guilabel: "บริษัท" (**ไม่ใช่ ** เป็น: guilabel: "บุคคล"), และ: guilabel: "ที่อยู่" และ: guilabel: "เลขประจำตัวผู้เสียภาษี" ต้องกรอกข้อมูลในช่อง

ลูกค้า¶

ตรวจสอบให้แน่ใจว่าแบบฟอร์มติดต่อลูกค้าของคุณกรอกอย่างถูกต้องเพื่อให้คุณใบแจ้งหนี้อิเล็กทรอนิกส์ที่ถูกต้อง:

  • ประเภทการติดต่อ:: guilabel: "ส่วนบุคคล": หรือ: guilabel: "บริษัท":

  • :: guilabel: "ประเทศ":

  • : guilabel: "เลขประจำตัวผู้เสียภาษี": เลขประจำตัวผู้เสียภาษีของบริษัทหรือนายทะเบียนบริษัท บัตรประจำตัวประชาชนสำหรับบุคคล

Note

คุณสามารถแก้ไขแบบฟอร์มการติดต่อของลูกค้าได้โดย:: menusesection: "บัญชี -> ลูกค้า -> ลูกค้า"

ผลิตภัณฑ์¶

ตรวจสอบให้แน่ใจว่าผลิตภัณฑ์ของคุณได้รับการกำหนดค่าอย่างถูกต้องสำหรับใบแจ้งหนี้อิเล็กทรอนิกส์ของคุณมีประสิทธิภาพ:

  • : guilabel: "ประเภทสินค้า": สินค้า วัสดุสิ้นเปลือง หรือบริการที่สามารถจัดเก็บได้

  • : guilabel: "หน่วยเมตริก": หากคุณใช้ Odoo ในสต็อกและเปิดใช้งานแล้ว:doc:`หน่วยการวัด<../../ investory_and_mrp / สินค้าคงคลัง / การจัดการผลิตภัณฑ์ / การกำหนดค่า / uom> `.

  • : ป้ายกำกับ: "บาร์โค้ด": ** GS1 * หรือ ** EGS ** บาร์โค้ด

  • : guilabel: "รหัสรายการ ETA" (ใต้: menuse ส่วน: แท็บ "การบัญชี"): ถ้าบาร์โค้ดไม่ตรงกับรหัสสินค้า ETA ของคุณ

Note

คุณสามารถแก้ไขผลิตภัณฑ์ของคุณได้โดย: ส่วนเมนู: บัญชี ->ลูกค้า --> สินค้า

การตรวจสอบสิทธิ์ USB¶

ทุกคนที่ต้องการเซ็นใบแจ้งหนี้อิเล็กทรอนิกส์ต้องใช้ USB เฉพาะตรวจสอบคีย์ของใบแจ้งหนี้ผ่าน ERP และส่งไปยังพอร์ทัล ETA

Note

คุณสามารถติดต่อ: ชื่อย่อ: "ETA (กรมสรรพากรอียิปต์)" หรือ "Egypt Trust"< https://www.egypttrust.com/ > ` _ รับคีย์ USB เหล่านี้

ติดตั้ง Odoo เป็น Local Agent บนคอมพิวเตอร์ของคุณ¶

เซิร์ฟเวอร์ท้องถิ่นของ Odoo ทำหน้าที่เป็นสะพานเชื่อมระหว่างคอมพิวเตอร์ของคุณกับ Odooฐานข้อมูลของโฮสติ้งออนไลน์

ดาวน์โหลด Odoo Community Installer จากหน้าhttps://www.odoo.com/page/download และในส่วนของคุณคอมพิวเตอร์

เลือก: guilabel: "โหมดพร็อกซี่ท้องถิ่น" เป็นประเภทการติดตั้ง

Note

การติดตั้ง Odoo นี้ทำงานเป็นเซิร์ฟเวอร์เท่านั้นและไม่มีการติดตั้งใด ๆแอป Odoo บนคอมพิวเตอร์ของคุณ

เมื่อติดตั้งเสร็จสมบูรณ์โปรแกรมติดตั้งจะแสดงสิทธิ์การเข้าถึง ** ของคุณโทเค็นสำหรับพร็อกซี่ท้องถิ่นของ Odoo ** คัดลอกโทเค็นและเก็บไว้ในที่ปลอดภัยเพื่อใช้ในภายหลัง

กำหนดค่าคีย์ USB¶

เมื่อติดตั้งพร็อกซีเซิร์ฟเวอร์ในเครื่องคอมพิวเตอร์ของคุณแล้วคุณสามารถเชื่อมโยงได้ใช้ฐานข้อมูล Odoo ของคุณ

  1. ไปที่: ส่วนเมนู: "การบัญชี -> การกำหนดค่า -> ไดรฟ์นิ้วหัวแม่มือ" และฮิต: guilabel:`Create'

  2. ป้อนชื่อ a:guilabel:"บริษัท", ป้อนข้อมูล:guilabel:ETA USB PIN`โดยผู้ให้บริการคีย์ USB ของคุณและในสิ้นสุด: อ้าง: การติดตั้งพร็อกซี่ในท้องถิ่น <Egypt / e-invoice-local-proxy>`,จากนั้นคลิกที่: guilabel: "บันทึก".

  3. คลิกที่: guilabel: "รับใบรับรอง".
