ไฟล์ PDF ที่สามารถพิมพ์ได้¶

พิมพ์เอกสาร PDF และฉลากที่เกี่ยวข้องกับการจัดส่งโดยอัตโนมัติใน Odoo ซึ่งมีรายละเอียดผู้รับพัสดุ เนื้อหา หรือคำแนะนำในการจัดการ

สามารถกำหนดค่า PDF ต่อไปนี้ให้พิมพ์เมื่อตรวจสอบการดำเนินการ สินค้าคงคลัง (เช่น การรับสินค้า การคัดเลือกสินค้า ใบส่งมอบสินค้า การตรวจสอบคุณภาพ):

  1. ใบส่งสินค้า

  2. ใบส่งคืน

  3. ฉลากสินค้าของรายการในการสั่งซื้อ

  4. ฉลากล็อตและหมายเลขซีเรียล

  5. ฉลากผู้ให้บริการ

  6. เอกสารการส่งออก

  7. สิ่งของในแพ็คเกจ

  8. ฉลากแพ็คเกจ

หากต้องการพิมพ์แบบฟอร์มเหล่านี้โดยอัตโนมัติ ให้ไปที่ แอปสินค้าคงคลัง ‣ การกำหนดค่า ‣ ประเภทการดำเนินงาน และเลือกประเภทการดำเนินการที่ต้องการ

ในแท็บ ฮาร์ดแวร์ ให้ทำเครื่องหมายที่ตัวเลือกที่ต้องการซึ่งมีอยู่ในส่วน พิมพ์เมื่อมีการตรวจสอบ เพื่อดาวน์โหลด PDF ของเอกสารที่เลือกโดยอัตโนมัติหลังจากตรวจสอบ ประเภทการดำเนินงาน สำหรับรายละเอียดเกี่ยวกับหน้าที่ของตัวเลือกกล่องกาเครื่องหมายแต่ละตัวเลือก ให้ข้ามไปที่ส่วนที่เกี่ยวข้อง

ใบส่งของ¶

ใบส่งของ จะประกอบด้วยรายละเอียดผู้รับและพัสดุ ซึ่งโดยปกติแล้วจะใส่ไว้ข้างใน (หรือแนบมากับ) พัสดุ

See also

  • รายการเลือกสินค้า

  • ฉลากติดตาม

หลังจาก เปิดใช้งานการตั้งค่าใบส่งสินค้า ในตัวเลือกการกำหนดค่าแท็บ ฮาร์ดแวร์ การคลิก ตรวจสอบ ที่ประเภทการดำเนินการที่ต้องการจะดาวน์โหลดใบส่งสินค้าในรูปแบบ PDF

ใบส่งสินค้าจะแสดงผลิตภัณฑ์ จำนวน หมายเลขอ้างอิงใบส่งสินค้า และน้ำหนักรวมของใบสั่งซื้อ

สลิปการส่งคืน¶

พิมพ์ ใบส่งคืนสินค้า เพื่อแนบไปกับการจัดส่งพัสดุคืนของลูกค้า ใบส่งคืนสินค้าจะระบุสินค้าที่ส่งคืน เชื่อมโยงกับใบสั่งขาย และรวมถึงรายละเอียดสินค้าและข้อมูลลูกค้า นอกจากนี้ยังสามารถรวมคำแนะนำการส่งคืนสินค้าเฉพาะสำหรับลูกค้าได้อีกด้วย

หลังจาก เปิดใช้งานการตั้งค่าใบส่งคืน ในตัวเลือกการกำหนดค่าแท็บ ฮาร์ดแวร์ การคลิก ตรวจสอบ ที่ประเภทการดำเนินการที่ต้องการจะดาวน์โหลด PDF ของใบส่งคืน

ใบส่งคืนสินค้าจะแสดงที่อยู่ผู้ส่งคืนของบริษัท พร้อมทั้งบาร์โค้ดสำหรับทั้งคำสั่งซื้อและการดำเนินการส่งคืนสินค้า

ฉลากสินค้า¶

พิมพ์ ฉลากสินค้า เพื่อแนบไปกับรายการในคำสั่งซื้อ โดยใส่ข้อมูลที่จำเป็น เช่น ชื่อสินค้า บาร์โค้ด และราคา

หลังจากที่นำทางไปที่ประเภทการดำเนินงานที่ต้องการ (แอปสินค้าคงคลัง ‣ การกำหนดค่า ‣ ประเภทการดำเนินงาน) ในแท็บ ฮาร์ดแวร์ ให้ทำเครื่องหมายที่ตัวเลือก ฉลากสินค้า

การดำเนินการดังกล่าวจะทำให้เมนูแบบเลื่อนลง พิมพ์ฉลากเป็น: ปรากฏขึ้น โดยฉลากผลิตภัณฑ์แต่ละรายการสามารถพิมพ์ได้ดังนี้:

  • 2 x 7 พร้อมราคา: PDF แสดงชื่อผลิตภัณฑ์ บาร์โค้ดและราคา โดยแสดงฉลากสินค้า 2 แถวและ 7 คอลัมน์ต่อหน้า

  • 4 x 7 พร้อมราคา: แสดงชื่อสินค้า บาร์โค้ดและราคา โดยจัดวางฉลากสินค้า 4 แถวและ 7 คอลัมน์ต่อหน้า

  • 4 x 12: แสดงชื่อสินค้าและบาร์โค้ด รองรับฉลากสินค้า 4 แถวและ 12 คอลัมน์ต่อหน้า

  • 4 x 12 พร้อมราคา: แสดงชื่อสินค้า บาร์โค้ด และราคา รองรับฉลากสินค้า 4 แถวและ 12 คอลัมน์ต่อหน้า

  • ฉลาก ZPL: พิมพ์ฉลากในภาษาการเขียนโปรแกรม Zebra (ZPL) ซึ่งประกอบด้วยชื่อสินค้าและบาร์โค้ด สามารถอ่านได้โดยเครื่องพิมพ์ Zebra เพื่อพิมพ์ฉลากโดยอัตโนมัติ

  • ป้าย ZPL พร้อมราคา: พิมพ์ฉลากใน Zebra Programming Language ซึ่งประกอบด้วยชื่อสินค้า บาร์โค้ดและราคา

Note

คุณสามารถพิมพ์ฉลากสินค้าด้วยตนเองจากคำสั่งจัดส่งได้ โดยคลิกปุ่ม พิมพ์ฉลาก

ฉลากล็อต/SN¶

พิมพ์ ฉลากล็อต/หมายเลขซีเรียล เพื่อแนบไปกับสินค้าในคำสั่งซื้อ โดยใส่ข้อมูลที่จำเป็น เช่น ชื่อสินค้า ล็อตหรือหมายเลขซีเรียลและบาร์โค้ด

หากต้องการพิมพ์ PDF นี้โดยอัตโนมัติ ให้ไปที่หน้าตัวเลือกประเภทการดำเนินการที่ต้องการ (แอปสินค้าคงคลัง ‣ การกำหนดค่า ‣ ประเภทการดำเนินงาน) จากนั้น ในแท็บ ฮาร์ดแวร์ ให้เลือกตัวเลือก ฉลากล็อต/หมายเลขซีเรียล

การดำเนินการดังกล่าวจะทำให้เมนูแบบเลื่อนลง พิมพ์ฉลากเป็น: ปรากฏขึ้น โดยฉลากผลิตภัณฑ์แต่ละรายการสามารถพิมพ์ได้ดังนี้:

  • 4 x 12 - หนึ่งชุดต่อล็อต/หมายเลขซีเรียล: PDF พร้อมฉลากสำหรับล็อต/หมายเลขซีเรียลเฉพาะตามลำดับ รวมทั้งชื่อผลิตภัณฑ์ ล็อต/หมายเลขซีเรียล และบาร์โค้ด เหมาะสำหรับ 4 แถวและ 12 คอลัมน์ต่อหน้า

  • 4 x 12 - หนึ่งชุดต่อหน่วย: PDF พร้อมฉลากที่ตรงกับจำนวนรายการ แสดงชื่อสินค้า ล็อต/หมายเลขซีเรียลและบาร์โค้ด เหมาะกับ 4 แถวและ 12 คอลัมน์ต่อหน้า

  • ฉลาก ZPL - หนึ่งฉลากต่อล็อต/SN: พิมพ์ฉลากในรูปแบบ Zebra Programming Language ซึ่งประกอบด้วยชื่อสินค้า ล็อต/หมายเลขซีเรียล และบาร์โค้ด

  • ฉลาก ZPL - หนึ่งฉลากต่อหน่วย: พิมพ์ฉลากพร้อมจำนวนรายการในรูปแบบ Zebra Programming Language ซึ่งประกอบด้วยชื่อสินค้า ล็อต/หมายเลขซีเรียล และบาร์โค้ด

ฉลากผู้ให้บริการ¶

หากต้องการพิมพ์ ฉลากผู้ให้บริการ โดยอัตโนมัติพร้อมที่อยู่ผู้รับ หมายเลขติดตาม และรายละเอียดผู้ให้บริการขนส่งสำหรับผู้ให้บริการขนส่งบุคคลที่สามรายใดรายหนึ่งโดยเฉพาะ ให้ดำเนินการตั้งค่าต่อไปนี้ให้เสร็จสมบูรณ์:

  1. ทำเครื่องหมายที่ช่อง ฉลากผู้ให้บริการ ใน การตั้งค่าประเภทการดำเนินงาน

  2. เชื่อมต่อเครื่องพิมพ์ ไปยังแอป IoT ของ Odoo

  3. กำหนดฉลากผู้ให้บริการให้กับเครื่องพิมพ์

  4. กำหนดค่าวิธีการจัดส่ง ประเภทฉลาก

กำหนดเครื่องพิมพ์¶

ดูรายละเอียดเกี่ยวกับการเชื่อมต่อเครื่องพิมพ์กับแอป IoT ของ Odoo ได้จากเอกสาร เชื่อมต่อเครื่องพิมพ์ เมื่อดำเนินการเสร็จสิ้น ให้กำหนดฉลากผู้ให้บริการให้กับเครื่องพิมพ์โดยไปที่ แอป IoT ‣ อุปกรณ์ และเลือกเครื่องพิมพ์ที่ต้องการ

ในแบบฟอร์มการกำหนดค่าเครื่องพิมพ์ ให้ไปที่แท็บ รายงานเครื่องพิมพ์ เพื่อกำหนดค่าประเภทของเอกสารที่เครื่องพิมพ์จะพิมพ์โดยอัตโนมัติ คลิก เพิ่มบรรทัด เพื่อเปิดหน้าต่างป๊อปอัป เพิ่ม: รายงาน ในแถบ ค้นหา... ให้พิมพ์ การจัดส่ง และเลือก ฉลากการจัดส่ง

Note

รายงาน เอกสารจัดส่ง มีไว้สำหรับ ส่งออกเอกสาร

หลังจากเพิ่มรายงาน ฉลากการจัดส่ง ในแท็บ รายงานเครื่องพิมพ์ แล้ว ตรวจสอบให้แน่ใจว่า ประเภทรายงาน ตรงกับประเภทของเครื่องพิมพ์ที่เชื่อมต่อ IoT

  • สำหรับเครื่องพิมพ์เลเซอร์ ให้ตั้งค่า ประเภทรายงาน เป็น PDF

  • สำหรับเครื่องพิมพ์ Zebra ให้ตั้งค่า ประเภทรายงาน เป็น ข้อความ

ประเภทฉลากผู้ให้บริการขนส่ง¶

ขั้นตอนต่อไป ให้ดำเนินการตั้งค่าให้เสร็จสิ้นสำหรับ ตัวเชื่อมต่อการจัดส่งของบุคคลที่สาม จากนั้นไปที่ แอปสินค้าคงคลัง ‣ การกำหนดค่า ‣ วิธีการจัดส่ง และเลือกวิธีการจัดส่งที่ต้องการ

ในแบบฟอร์มกำหนดค่าวิธีการจัดส่ง ในแท็บ การกำหนดค่า [ชื่อผู้ให้บริการ] ตรวจสอบให้แน่ใจว่า รูปแบบฉลาก ตรงกับ ประเภทรายงานที่กำหนดไว้ก่อนหน้านี้:

  • สำหรับเครื่องพิมพ์เลเซอร์ ให้ตั้งค่า รูปแบบฉลาก เป็น PDF

  • สำหรับเครื่องพิมพ์ Zebra ให้ตั้งค่า รูปแบบฉลาก เป็น ZPL2

ตัวอย่างฉลากผู้ให้บริการ¶

หลังจากตรวจสอบการทำงานแล้ว ฉลากผู้ให้บริการจะถูกสร้างขึ้นในช่องแชท และพิมพ์โดยใช้เครื่องพิมพ์ที่เชื่อมต่อ IoT

เอกสารการส่งออก¶

เอกสารการส่งออก ที่ศุลกากรต้องการในการจัดส่งพัสดุจากประเทศหนึ่งไปยังอีกประเทศหนึ่ง สามารถพิมพ์ใน Odoo ได้โดยอัตโนมัติ โดยทำตามขั้นตอนเหล่านี้:

  1. ทำเครื่องหมายที่ช่องทำเครื่องหมาย ส่งออกเอกสาร ใน การตั้งค่าประเภทการดำเนินการ

  2. เชื่อมต่อเครื่องพิมพ์ ไปยังแอป IoT ของ Odoo

  3. กำหนดเอกสารส่งออกไปยังเครื่องพิมพ์

กำหนดเครื่องพิมพ์¶

คล้ายกับ คำแนะนำการกำหนดเครื่องพิมพ์สำหรับฉลากผู้ให้บริการ หลังจากเชื่อมต่อเครื่องพิมพ์ที่เข้ากันได้กับแอป Odoo IoT แล้ว ให้ไปที่ แอป IoT ‣ อุปกรณ์ และเลือกเครื่องพิมพ์ที่ต้องการ

ในแบบฟอร์มการกำหนดค่าเครื่องพิมพ์ ให้ไปที่แท็บ รายงานเครื่องพิมพ์ แล้วคลิก เพิ่มบรรทัด ในหน้าต่างป๊อปอัป เพิ่ม: รายงาน ที่ปรากฏขึ้น ให้เพิ่มรายงาน เอกสารการจัดส่ง เพื่อกำหนดเอกสารส่งออกไปยังเครื่องพิมพ์

สิ่งของในแพ็คเกจ¶

ไฟล์ PDF สิ่งของในแพ็คเกจ ประกอบด้วยบาร์โค้ดของแพ็คเกจ วันที่บรรจุ พร้อมทั้งรายการผลิตภัณฑ์ที่บรรจุและปริมาณ

หากต้องการพิมพ์แบบฟอร์มนี้โดยอัตโนมัติ ให้ไปที่ แอปสินค้าคงคลัง ‣ การกำหนดค่า ‣ ประเภทการดำเนินการ และเลือกประเภทการดำเนินการที่ต้องการ จากนั้นไปที่แท็บ ฮาร์ดแวร์ และทำเครื่องหมายที่ช่อง สิ่งของในแพ็คเกจ

Important

If the option is not available, enable the Packages feature, by going to Inventory app ‣ Configuration ‣ Settings, ticking the Packages checkbox, and clicking Save.

หลังจากเปิดใช้งานฟีเจอร์ในแท็บ ฮาร์ดแวร์ แล้ว การตรวจสอบการทำงานจะพิมพ์ PDF ของสิ่งของในแพ็คเกจ

ฉลากแพ็คเกจ¶

คุณสามารถกำหนดค่า ฉลากแพ็คเกจ ที่แสดงบาร์โค้ดและวันที่บรรจุภัณฑ์ของบรรจุภัณฑ์ให้พิมพ์ได้โดยคลิกปุ่ม ใส่ในแพ็ค

Important

The Put in Pack button is available only when the Packages feature is enabled in Inventory app ‣ Configuration ‣ Settings.

หลังจากเปิดใช้งานแล้ว ปุ่ม ใส่ในแพ็ค จะพร้อมใช้งานในการดำเนินการคงคลังทั้งหมด (เช่น การรับสินค้า การคัดเลือกสินค้า การโอนภายใน ใบสั่งส่งมอบสินค้า ฯลฯ)

หากต้องการพิมพ์ฉลากแพ็กเกจโดยอัตโนมัติเมื่อคลิกปุ่ม ใส่ในแพ็ค ให้ไปที่ แอปสินค้าคงคลัง ‣ การกำหนดค่า ‣ ประเภทการดำเนินการ เลือกประเภทการดำเนินการที่ต้องการ แล้วทำเครื่องหมายที่ช่อง ฉลากแพ็คเกจ ในแท็บ ฮาร์ดแวร์ สามารถพิมพ์ฉลากในรูปแบบไฟล์ PDF หรือ ZPL ตามที่กำหนดไว้ในฟิลด์ พิมพ์ฉลากเป็น
